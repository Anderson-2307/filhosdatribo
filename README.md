# Filhos da Tribo — Site Oficial

Site institucional da marca artesanal **Filhos da Tribo**, sediada em Salvador, Bahia, Brasil.

## Como usar

1. Clone o repositório
2. Coloque todos os arquivos de imagem dentro da pasta `imagens/`
3. Abra o `index.html` no navegador — nenhuma dependência externa além do Google Fonts

## Tecnologias

- HTML5 semântico
- CSS3 puro (Grid, Flexbox, variáveis CSS, animações)
- JavaScript vanilla (menu mobile, troca de imagens, scroll reveal)
- Google Fonts: Playfair Display, Dancing Script, Inter

## Publicação no GitHub Pages

1. Faça o push para o branch `main`
2. Vá em **Settings → Pages**
3. Source: `Deploy from a branch` → branch `main` → pasta `/` (root)
4. O site ficará disponível em `https://seu-usuario.github.io/filhosdatribo`
